# Charge Alert
An Android app which plays a notification sound when a charger is plugged in or unplugged. You can pick which sound it plays, which can be different for plugging in and unplugging, from a dropdown of all the notification sound variants on your phone.

<img width="300" alt="screenshot" src="screenshot.png" />

# Installation
1. Download the .apk file from the latest release onto your phone
2. Go to your Files app > Downloads > and open the .apk file you just downloaded
3. Tap Settings. (It will tell you that your phone isn't currently allowed to install apps from this source. So go to Settings, where you will be able to change that)
4. Enable "allow from this source"
5. Tap Install
6. Tap the tiny text saying "More details" when it says "App scan recommended"
7. Tap Install without scanning

The app should now exist in your "All apps" screen!
